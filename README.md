# TIL-RESTART (Today I Learned)

## 📚 개요

이 저장소는 매일 학습한 내용을 기록하는 공간입니다.  

주요 목표는 다음과 같습니다:
- 📈 **지속적인 성장** : 작더라도 꾸준히 기록  
- 🧩 **깊이 있는 이해** : 단순한 암기가 아닌 근본 원리 탐구  
- 💬 **체계적인 정리** : 프롬프트를 이용해 학습 내용을 구조화하고 명료하게 기록  

이 저장소는 ChatGPT를 활용한 **1회 정리 프로세스**를 포함하며,  
지속적으로 학습한 내용을 체계적으로 관리하는 것을 목표로 합니다.
